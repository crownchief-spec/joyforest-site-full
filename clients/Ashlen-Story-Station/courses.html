用大腦最喜歡的方式學英文

我們拒絕填鴨式默背死學科。在常態教室裡，英文就是孩子完成一項項硬核專題的真實工具。

STAGE 01 // 2.5歲–5歲

繪本故事課（互動式聽力先行）

從故事聽讀出發，用繪本自然灌溉孩子的耳朵。拒絕點讀筆的單向冷冰刺激，透過高質量的互動式聽力，在幼兒大腦中建立最純粹的「英文聲音資料庫」。

#故事教學 #互動式聽力
STAGE 02 // 5歲–7歲

雙軌拼讀與口說課（44語音推導）

把握拼讀黃金期，拒絕死背字母視覺符號，從 44 個英文語音（Sounds）切入，用聲音無痛對接字形。口說教育拒絕背誦公式句型，引導孩子有內容地說出想法。

#自然拼讀 #44個語音 #口說表達
STAGE 03 // 國小兒美進階課

日常 PBL 專題任務（真實語言輸出）

每學期全面內嵌專題式任務學習（PBL）。孩子學會自發查找全英文資料、組織資訊並上台簡報發表。化身「小小文化記者」與海外兒童線上全英文對話交鋒。
